## 자꾸 잊어버리는 C언어 문법

### <입출력 문법>
||입력시(scanf문)|출력시(printf문)|
|--|--|--|
|정수형 int|%d|%d|
|문자형 char|%c|%c|
|실수형 float|%f|%f|
|실수형 double|%lf|%f|
>실수형 scanf문 작성할 때, %lf에서 소문자 L임에 주의!

### <소수점으로 나타내기>
ex) 소수점 2자리까지 나타내기 -> "%.2f"
